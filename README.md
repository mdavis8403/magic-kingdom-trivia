# Magic Kingdom Trivia

Magic Kingdom Trivia is a native Android TV trivia experience built with Kotlin and Jetpack Compose for TV.

## Highlights

- D-pad friendly category selection and answer flow
- Curated Magic Kingdom question bank with themed categories
- Score tracking, streaks, and end-of-round recap
- Pure Kotlin game engine with unit tests

## Build

1. Ensure a JDK and Android SDK are installed.
2. Set `ANDROID_HOME` or create a `local.properties` file with `sdk.dir=...`.
3. Run `./gradlew assembleDebug test`.
